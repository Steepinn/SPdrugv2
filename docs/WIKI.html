SPdrug — вики по плагину

Плагин для Paper 1.21: лаборатория (синтез), теплица (сушка), полевые культуры, препараты с эффектами, сигареты. Источник истины — src/main/resources/config.yml и код ru.spdrug.

Версия jar: смотри build.gradle (version = …).

Требования и установка

  • Сервер: Paper (API 1.21, в проекте paper-api:1.21.1-R0.1-SNAPSHOT).
  • Java 21 (build.gradle).
  • Сборка: в корне проекта gradlew build → build/libs/SPdrug-<версия>.jar.
  • Jar в plugins/, перезапуск сервера.
  • Первый запуск создаёт plugins/SPdrug/config.yml. После правок: /spdrug reload (нужно spdrug.admin).

Предметы и блоки

Настройки: blocks.*, item-materials.*, items.* в config.yml.

  • Лаборатория — ПКМ по блоку ставит лабораторию (по умолчанию SMOKER), GUI синтеза.
  • Теплица — компостер + зона farm.horizontal-radius, farm.vertical-range.
  • Агрочасы — ПКМ по созревшему папоротнику: прогресс и буст.
  • Сырьё / сушёный сбор — предметы с метками плагина; материалы по умолчанию DRIED_KELP / NETHER_WART должны совпадать с recipe у препаратов с сушкой в первом слоте.
  • Препарат — ПКМ: эффекты, титул, боссбар, плохая реакция, передоз.
  • Сигарета — пустая из бумаги, заряд табачным продуктом; затяжка даёт короткий реген (cigarette.effect-ticks).

Игровой цикл

  1. Особые семена (трава или /spdrug give seeds).
  2. Посадка на грядку (FARMLAND) → папоротник; светокамень на Y+2 ускоряет и бустит урожай.
  3. Слом созревшего → сырьё cannabis / cocaine / tobacco.
  4. В зоне теплицы ПКМ по компостеру с сырьём → сушка (мини-игра).
  5. Сушёный сбор + химия в лаборатории → препарат или промежуточные компоненты.

Пшеница в зоне теплицы при полном возрасте даёт сырьё типа каннабис — см. SPdrugListener.

Лаборатория

  • Реагенты слева направо, порядок как в рецепте.
  • Матч по recipe-keys: CHEM:<id> (предмет химии) и MAT:MATERIAL.
  • first-slot-must-be-dried-batch: true — первый слот только сушёный сбор плагина, не обычный предмет того же типа. required-dried-type — тип dried_batch:<тип>.
  • recipe — проверка материала в слоте; длина и порядок = как в GUI.
  • После старта — удержание стабильности (ЛКМ по лаборатории). Ошибка или провал → уничтожение лаборатории (laboratory.explosion-*).
  • Несколько полных наборов в слотах → пропорциональный выход, остаток возвращается.

Теплица и сушка

  • Зона: квадрат вокруг компостера; при horizontal-radius: 2 сторона 5.
  • Сушка: сырьё в основной или второй руке, ПКМ по компостеру; длительность farm.drying-duration-ticks.

Полевые культуры (семена)

  • Посадка только на грядку (FARMLAND); иначе messages.field-herb-seed-need-farmland.
  • До созревания при ломании падает 1 особое семя того же типа (без сырья). В зрелости — 1 или 2 сырья при бусте светокамнем.
  • Отдельных предметов без типа нет: пшеница в теплице и запасные пути дают каннабис; give raws / give dried — только ветки из items.herb-raw-types / items.dried-batch-types.
  • farm.field-herb-grow-duration-seconds — базовое время созревания.
  • farm.field-herb-grow-speed-multiplier — чем больше, тем быстрее (время ÷ множитель).
  • Светокамень на Y+2 от посадки — ускорение и бонус к количеству при сборе в зрелости.

Препараты и сигареты

  • В drugs: positive-effects, negative-effects, bad-trip-chance; передоз — drug-use.overdose.
  • drug-use: титул, боссбар, урон плохой реакции (bad-trip-damage-min/max), вспышки (screen-flash-count, интервал в тиках; 0 — выкл).
  • Сигарета: cigarette.max-uses, cigarette.effect-ticks — длительность REGENERATION I за затяжку.

Частые проблемы

Лаборатория взорвалась

  • Сверь recipe-keys и recipe: длина, порядок, первый слот — сушёный сбор с меткой, не обычный NETHER_WART.
  • Для кокаина/табака — required-dried-type = cocaine / tobacco.
  • Химия должна быть синтезирована (PDC), не «голый» снежок.

После обновления jar что-то не так

  • Сравни plugins/SPdrug/config.yml с версией из jar/репозитория (новые ключи field-herb-*, drug-use.bad-trip-* и др.).

Книга устарела

  • Выдай новую: /spdrug give book или give all.
