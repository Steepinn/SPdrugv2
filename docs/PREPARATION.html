SPdrug — приготовление и действие препаратов

Справочник по умолчанию из config.yml. В лаборатории реагенты кладутся слева направо строго по порядку. Ошибка рецепта — реактор уничтожается.

Разделы: Обозначения · Цепочка от семян · Базовая химия · Составная химия · Финальные препараты · Сигарета

Обозначения

  • CHEM:id — предмет химии плагина (обычно выглядит как снежок с меткой).
  • MAT:ИМЯ — обычный предмет Minecraft с таким материалом.
  • Сушёный сбор — не обычный нарост: предмет с меткой плагина после сушки на теплице. В рецепте материал слота часто NETHER_WART, но нужен именно сушёный сбор нужного типа (cannabis, cocaine, tobacco).

Книгу рецептов в игре можно взять командой /spdrug book (право spdrug.book, по умолчанию у всех).

Общая цепочка

  1. Семена (каннабис / кока / табак) → посадка на землю → папоротник. Светокамень на +2 блока по Y над посадкой ускоряет рост и даёт бонус урожая.
  2. Сбор → сырьё соответствующего типа.
  3. Теплица: ПКМ по компостеру с сырьём в руке → сушка → сушёный сбор.
  4. Лаборатория: по шагам крафтить chemicals, затем финальный рецепт из секции drugs.

Сигарета

Пустая сигарета крафтится из 3 бумаги в один ряд (см. crafting.cigarette). Затем в верстаке объединить с табачным продуктом (препарат nicotine).

Затяжка (ПКМ): короткий эффект регенерации I на cigarette.effect-ticks тиков; число затяжек — cigarette.max-uses.

При употреблении любого препарата дополнительно действуют настройки drug-use: титул, боссбар, вспышки слепоты, передоз при частом приёме (drug-use.overdose). Точные числа при смене конфига смотри в своём config.yml.
