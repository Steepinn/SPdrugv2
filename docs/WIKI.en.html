Requirements and installation

  • Server: Paper (API 1.21, project uses paper-api:1.21.1-R0.1-SNAPSHOT).
  • Java 21 (build.gradle).
  • Build: gradlew build in project root → build/libs/SPdrug-<version>.jar.
  • Place jar in plugins/, restart server.
  • First run creates plugins/SPdrug/config.yml. After edits: /spdrug reload (needs spdrug.admin).

Commands

/spdrug book — any player with spdrug.book. Other give and reload — spdrug.admin only. Items go to inventory (overflow drops).

Items and blocks

Settings: blocks.*, item-materials.*, items.* in config.yml.

  • Laboratory — right-click to place lab block (default SMOKER), synthesis GUI.
  • Greenhouse — composter + zone farm.horizontal-radius, farm.vertical-range.
  • Growth watch — right-click grown field fern: progress and boost.
  • Raw / dried batch — tagged items; default materials DRIED_KELP / NETHER_WART must match recipe for drugs with dried slot rules.
  • Drug — right-click: effects, title, boss bar, bad trip, overdose.
  • Cigarette — empty from paper, charge with tobacco product; puff gives short regen (cigarette.effect-ticks).

Gameplay loop

  1. Special seeds (grass or /spdrug give seeds).
  2. Plant on farmland → fern; glowstone at Y+2 speeds growth and boosts mature harvest.
  3. Break mature plant → raw cannabis / cocaine / tobacco.
  4. In greenhouse zone, right-click composter with raw → drying (minigame).
  5. Dried batch + chemistry in lab → drug or intermediates.

Wheat in the greenhouse at full age drops cannabis raw — see SPdrugListener.

Laboratory

  • Reagents left to right, order matches recipe.
  • Match via recipe-keys: CHEM:<id> (chemistry item) and MAT:MATERIAL.
  • first-slot-must-be-dried-batch: true — slot 1 must be plugin dried batch, not a plain item of same type. required-dried-type — type dried_batch:<type>.
  • recipe — material check per slot; length and order = GUI layout.
  • After start — hold stability (LMB on lab). Wrong recipe or failure → lab destroyed (laboratory.explosion-*).
  • Multiple full sets in slots → scaled output, remainder returned.

Greenhouse and drying

  • Zone: square around composter; with horizontal-radius: 2 side is 5.
  • Drying: raw in main or off hand, right-click composter; duration farm.drying-duration-ticks.

Field crops (seeds)

  • Plant only on farmland (FARMLAND); otherwise messages.field-herb-seed-need-farmland.
  • Before maturity, breaking drops 1 special seed of the same type (no raw). When mature, 1 or 2 raw with glowstone boost.
  • No untyped “generic” raw/dried items: greenhouse wheat and fallbacks use cannabis; give raws / give dried only list items.herb-raw-types / items.dried-batch-types.
  • farm.field-herb-grow-duration-seconds — base grow time.
  • farm.field-herb-grow-speed-multiplier — higher = faster (time ÷ multiplier).
  • Glowstone Y+2 above plant — speed and mature yield bonus.

Drugs and cigarettes

  • In drugs: positive-effects, negative-effects, bad-trip-chance; overdose — drug-use.overdose.
  • drug-use: title, boss bar, bad trip damage (bad-trip-damage-min/max), flashes (screen-flash-count, interval ticks; 0 = off).
  • Cigarette: cigarette.max-uses, cigarette.effect-ticks — REGENERATION I duration per puff.

Troubleshooting

Lab exploded

  • Match recipe-keys and recipe: length, order, first slot is tagged dried batch, not plain NETHER_WART.
  • For cocaine/tobacco — required-dried-type = cocaine / tobacco.
  • Chemistry must be synthesized (PDC), not a bare snowball.

Odd behaviour after jar update

  • Compare plugins/SPdrug/config.yml with jar/repo (new keys field-herb-*, drug-use.bad-trip-*, etc.).

Book is outdated

  • Get a new one: /spdrug give book or give all.
