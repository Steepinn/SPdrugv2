# SPdrug

Плагин для **Paper 1.21**: лаборатория, теплица, полевые культуры, синтез, препараты.

**Автор:** Steepin · **SteepStudio**  
**Лицензия:** [MIT](LICENSE)  

Сборка: `gradlew build` → `build/libs/SPdrug-*.jar`  
Язык интерфейса: в `config.yml` → `general.language: en` (переводы в jar: `lang/en.yml`), затем `/spdrug reload`.  
Документация: `docs/WIKI.md`, `docs/WIKI.html`, `docs/PREPARATION.html`
